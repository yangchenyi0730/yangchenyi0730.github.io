常用命令

日常情况只需要使用三个即可：

  • hexo new “titleName”
  • hexo c; hexo g; hexo s
  • hexo c; hexo g; hexo d
  • hexo c
    • 清除缓存
  • hexo g
    • 渲染md文件为html文件
  • hexo s
    • 启动本地服务，调试查看
  • hexo d
    • 部署内容，将博客内容部署到github
  • hexo new “titleName”
    • 新建文章
  • 命令组合：
    • hexo c; hexo g; hexo s
      • 本地查看新发布内容
    • hexo c; hexo g; hexo d
      • 远程推送新发布内容
